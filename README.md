# RAG-Based_Security_Auditor-
Automated CVE and OWASP Top 10 Code Compliance using Retrieval-Augmented Generation
This project develops a RAG-based system that connects a Large Language Model to a curated knowledge base containing the latest OWASP Top 10 standards and CVE databases.

When a developer submits code, the system retrieves relevant security patterns and vulnerability descriptions to cross-check the code for security flaws — grounding AI analysis in real, updated documentation.
